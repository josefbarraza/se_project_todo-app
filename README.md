# Todoey App

Todoey is a user-friendly task management application that enables users to create and organize to-do lists. The application allows users to add tasks, assing due dates, and track task completion through interactive checkbox functionality. It was designed wih a focus on usability, responsive functionality, and efficient task organiztion.

## Functionality

The application includes form validation functionality that verifies user input entered into the "Add New ToDo" form before a task is submitted. Additionally, a delete feature was implemented to allow users to remove completed tasks from the list.

## Technology

This project was built using HTML, CSS, and JavaScript. It uses techniques such as form validation, event listeners, and DOM manipulation to create an interactive and user-friendly to-do list application. Responsive design was also used to help the application work across different screen sizes.
